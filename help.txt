-- Comandos GIT

	-Fluxo de salvar no git
	
		git status -- opcional,para verificar os arquivos que tu mudou.
		git add [ . ; NOME_DO_ARQUIVO] -- adiciona todos ( "." ) ou um arquivo para a lista de arquivos que devem ser salvos.
		git commit -m "TEU_COMENTARIO_ESPECIFICANDO_QUE_FOI_FEITO" -- salva as alteracoes feitas no seu local.
		git push -- salva as alteracoes no git/empurra os arquivos alterados para o git.
	
	-Comandos Uteis
	
		git checkout . -- reseta todas as tuas alteracoes em todos os arquivos alterados ate o ultimo commit.
		git add --all  -- dependendo da tua versao do git, ele nao aceita o argumento "." para adicionar arquivos deletados, nesses casos deve passar o argumento --all.
		git pull -- procura e puxa alteracoes nos arquivos
